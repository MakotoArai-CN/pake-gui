## Development

```bash
# Install dependencies
bun install

# Start development server
bun run tauri dev

# Build for production
bun run tauri build
```

## Requirements

- Node.js or Bun.js
- Rust
- Pake CLI
- Visual Studio Build Tools (Windows only)

The application will guide you through installing missing dependencies.
